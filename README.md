# habit_tracker_app
A practical web application built with **Python**, **Streamlit**, and **Pandas** designed to help users build discipline, log routines, and gain insights into their daily consistency.

### Main Features
* **Habit Management:** Easily create, update, and organize daily, weekly, or monthly habits.
* **Streak & Stats Tracking:** Automated calculations for current and best streaks, along with total completion rates.
* **Smart Recommendations:** Behavior design tips that suggest the best routines to follow based on user consistency.
